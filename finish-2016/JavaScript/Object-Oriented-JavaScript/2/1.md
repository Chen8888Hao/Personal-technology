undefined 当我们视图访问一个不存在的变量的时候，就会得到一个特殊值：undefined。使用一个未初始化的变量也会这样。js会自动将变量在初始化之前设定为undefined.

null 这是另一种只包含一个值的特殊数据类型。所谓的null值，通常是指没有值、空值，不代表任何东西

null和undefined最大的区别在，被赋予null的变量通常被认为已定义的，只不过它不代表任何东西
