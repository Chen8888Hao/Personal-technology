我们拷贝某个对象或者把它传递给某个函数时，往往传递的都是该对象的引用。因此我们在引用上所做的任何改动，实际上都会影响他所引用的原对象。

---

内建对象：

数据封装类对象: Object Array Boolean Number String

工具类对象: Math Date RegExp
